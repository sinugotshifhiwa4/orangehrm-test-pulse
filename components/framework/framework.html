Test Automation Framework
The repository that runs the tests and feeds every number on this dashboard.

A Playwright + TypeScript end-to-end automation framework for OrangeHRM, built around reusable page objects, a layered fixture architecture, stage-based environment resolution, and CI-ready execution. Each CI run publishes its results, which Test Pulse reads and visualises.

Tech stack
Playwright TypeScript Winston Luxon Faker dotenv cross-env ESLint Prettier Husky lint-staged Ortoni Report
Key features
  • Layered fixture architecture (base · UI · API)
  • Page Object Model for maintainable selectors
  • Stage-based environment resolution (QA · UAT · preprod)
  • Tag-based test filtering & worker parallelisation
  • Authentication setup & login orchestration
  • Structured logging with Winston
  • Pre-commit quality gates via Husky + lint-staged
Repository structure
src/        Framework core — configuration, layers, utilities
fixtures/   Test fixtures, organised by layer
tests/      Test suites by layer (ui · api · db)
scripts/    Test execution scripts
envs/       Environment configuration files
docs/       Framework documentation
